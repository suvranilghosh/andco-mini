# Andco Mini

A document request tracker for paralegals to monitor, filter, and update the status of outstanding case records.

## Stack

- [Next.js 16](https://nextjs.org/) — App Router, React Server Components
- [shadcn/ui](https://ui.shadcn.com/) — component library
- [Tailwind CSS v4](https://tailwindcss.com/) — styling
- [Radix UI](https://www.radix-ui.com/) — accessible primitives (Select, Drawer, Tooltip)
- [Phosphor Icons](https://phosphoricons.com/) — icons
- [next-themes](https://github.com/pacocoursey/next-themes) — light/dark mode

## Getting Started

```bash
npm i
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
